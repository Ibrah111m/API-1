Välkommen till min server

Denna server hanterar användarinformation och autentisering genom följande funktioner:

  • GET /users: Hämtar alla användare (kräver giltig JWT-token).
  • GET /users/:id: Hämtar en specifik användare baserat på ID.
  • POST /users: Skapar en ny användare med användarnamn, lösenord, namn och e-post.
  • PUT /users/:id: Uppdaterar informationen för en befintlig användare (kräver giltig JWT-token).
  • POST /login: Autentiserar användaren och genererar en JWT-token.

För att testa och använda dessa funktioner, kan du använda verktyg som Insomnia för att skicka HTTP-förfrågningar. För att hantera databasen kan du använda MySQL för att utföra SQL-querys.

Lösenorden lagras som hashade värden i databasen för säkerhetens skull.
